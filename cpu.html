CPU

A cpu or central processing unit is the main processor powering your computer. A modern desktop PC will use either an Intel or AMD cpu

Intel

Intel CPUs

  • 13th generation CPUs have up 8 performance and 16 efficiency cores
  • These CPUs use the Z790/H770/B760 chipset motherboards

AMD

AMD CPUs

  • 7000 series CPUs have up to 16 cores
  • These CPUs use the X670E/X670/B650 chipset motherboards
